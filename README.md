# EasyLanguage Trading Strategies

This repository will contain **EasyLanguage code snippets** to analyze possible usage approaches and logic patterns.

The code snippets will be accompanied by **brief technical analyses**.

The revisited snippets come from:
- Proprietary implementations
- External sources (e.g., trading research)

## Structure

- `/src` – code snippets (.EL, .txt)
- `/analysis` – brief reports and comments on the snippets

## Disclaimer

The code is for demonstrative and educational purposes only. It does not constitute investment advice.
